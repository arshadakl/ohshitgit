ഫയലുകൾ മാറ്റി എന്ന് അറിയാം, പക്ഷേ `diff` ഒഴിഞ്ഞതാണ് കാണുന്നത്, ഒരുപക്ഷേ `add` ചെയ്ത് staging-ൽ ഇട്ടതായിരിക്കും, ഒരു special flag ഉപയോഗിക്കണം.

```git
git diff --staged
```
